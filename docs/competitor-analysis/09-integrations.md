# Integrations Hub

**Screenshot:** `33-integrations.png`

Route: `/integrations`

---

## Overview

- Headline: **"Integrations Hub"**
- Subtitle: *"Browse supported integrations to extend your bundle capabilities."*
- **"Request Integration"** button — allows merchants to suggest new integrations

---

## Supported Integrations

### Pre-orders, Pickup & Delivery
| App | Integration Description |
|-----|------------------------|
| **Stoq** | Pre-order out-of-stock items within your bundles |
| **Zapiet** | Schedule store pickup & delivery for bundle orders |

### Subscriptions
| App | Integration Description |
|-----|------------------------|
| **Skio** | Add subscription selling plans to bundled products |
| **Appstle** | Enable subscribe-and-save options on your bundles |
| **Bold** | Set up recurring bundle subscriptions via Bold |

### Reviews
| App | Integration Description |
|-----|------------------------|
| **Judge.me** | Display star ratings and reviews on your bundles |

### Page Builders
| App | Integration Description |
|-----|------------------------|
| **PageFly** | Create custom landing pages to showcase bundles |
| **GemPages** | Build high-converting pages for your bundle store |

### Checkout
| App | Integration Description |
|-----|------------------------|
| **Gokwik** | Streamlined Indian checkout experience for bundles |
| **Shopflo** | Optimized Indian checkout flow with bundle support |

---

## Key Observations

- Each integration has a **"View Setup"** button — suggests a setup guide/doc rather than a UI-level configuration
- The subscription integrations (Skio, Appstle, Bold) are the same 3 shown in the Bundle Editor → Subscriptions panel, confirming the integrations hub is a documentation hub, not an in-app connection manager
- The **India-specific checkout integrations** (Gokwik, Shopflo) reveal a strong Indian merchant customer base — aligning with the Skai Lama brand being India-headquartered
- The **"Request Integration"** CTA is both a product feedback channel and a list-building tool (merchants who request get notified when the integration ships)
- Notably absent: Klaviyo/email, Yotpo/reviews, Recharge (alternative subscription), Tapcart (mobile) — potentially gaps vs. some merchant needs
